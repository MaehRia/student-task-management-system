# Student Task Management System

This project was developed as part of the Software Engineering course.
It provides a web-based platform for students to manage academic tasks, monitor deadlines, and track task completion.

## Features
- User registration and authentication
- Create, update, and delete tasks
- Task list and status tracking
- SQLite database storage

## How to Run
1. Create and activate a virtual environment
2. Install required dependencies
3. Run `python run.py`
4. Open `http://127.0.0.1:5000` in a browser
